# Images Directory

This directory contains all images for the OptiCore OS documentation.

## Required Images

### Logo
- `logo.png` - Main project logo (200x200px recommended)

### Screenshots
- `screenshots/ame-wizard-main.png` - AME Wizard main interface
- `screenshots/powershell-admin.png` - PowerShell running as Administrator
- `screenshots/script-execution.png` - Script execution process
- `screenshots/optimization-result.png` - Optimization completion notification
- `screenshots/before-after.png` - Before and after comparison

### Diagrams
- `flowchart.png` - Installation flowchart
- `architecture.png` - System architecture diagram
- `benchmark-results.png` - Benchmark comparison chart

## Image Guidelines

1. **Format**: PNG or SVG preferred
2. **Background**: Transparent or matching documentation theme
3. **Size**: 
   - Logo: 200x200px
   - Screenshots: 1280x720px or similar aspect ratio
4. **Style**: Consistent with project branding (blue/green accents)
